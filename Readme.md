### Структура микросервиса:
* Вход
* API (документация)
* Логика
* БД
* Выход (json)
* Тесты

### Общие требования к микросервису:
* Максимально автономный (независимый).
* Выполняет только базовую функцию.
* Содержит всегда поддерживаемую минимальную версию.
* В отправленых данных содержится версия микросервиса.
